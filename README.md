# Spam Mail Prediction using Machine Learning

## Project Description
This project predicts whether an email message is spam or not using machine learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Algorithm Used
Naive Bayes Classifier

## Dataset
The dataset contains email messages labeled as spam or ham.

## How to Run the Project

1. Install dependencies:
pip install -r requirements.txt

2. Run the program:
python spam_classifier.py

3. Enter a message when asked.

## Output
The model predicts whether the email is Spam or Not Spam.
